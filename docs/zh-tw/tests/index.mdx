這些測試驗證開發容器的主要子系統在執行時期能否協同運作。它們並未內建於 Docker 映像檔中——請在執行中的容器內執行這些測試以驗證其功能。

## 可用測試

| 測試 | 驗證內容 |
| ---- | ---- |
| [螢幕錄製](/devcontainer/tests/screen-recording/) | Xvfb 顯示、有介面的 Chromium（Playwright）和 ffmpeg 錄製能產生有效的 MP4 檔案 |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | Chrome 符號連結、無頭模式修補和 MCP 導航能產生有效的螢幕截圖 |
