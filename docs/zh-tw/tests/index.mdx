這些測試驗證 devcontainer 的主要子系統在執行時期能否協同運作。它們並未內建於 Docker 映像中——請在執行中的容器內運行這些測試，以驗證其功能。

## 可用測試

| 測試 | 驗證內容 |
| ---- | ---- |
| [螢幕錄製](/devcontainer/tests/screen-recording/) | Xvfb 顯示、有界面 Chromium（Playwright）及 ffmpeg 錄製能產生有效的 MP4 |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | Chrome 符號連結、無頭模式修補及 MCP 導覽能產生有效的螢幕截圖 |
