คอนเทนเนอร์จะรันชุดจอแสดงผลเสมือน (Xvfb + x11vnc + noVNC + fluxbox) เพื่อให้คุณสามารถดู AI agents ควบคุมเบราว์เซอร์และเข้าแทรกแซงเมื่อจำเป็น — เช่น เพื่อจัดการกับหน้าล็อกอินหรือข้อความยืนยัน MFA

## การเชื่อมต่อ

1. เปิด `http://localhost:6080/vnc.html` ในเบราว์เซอร์ของคุณและคลิก **Connect**
2. คุณจะเห็นเดสก์ท็อป fluxbox แบบเบา — นี่คือจอแสดงผลเสมือนของคอนเทนเนอร์

AI agents ที่ใช้ `@playwright/mcp` ในโหมด headed จะเรนเดอร์ Chromium ลงบนจอแสดงผลนี้ คุณสามารถคลิกและพิมพ์โดยตรงในหน้าต่าง noVNC เพื่อเข้าแทรกแซงได้ตลอดเวลา — ไม่จำเป็นต้องหยุด/เริ่มใหม่

## การเปิดเบราว์เซอร์ด้วยตนเอง

วิธีเปิดเบราว์เซอร์ด้วยตนเองภายในคอนเทนเนอร์:

```bash
chromium --no-sandbox &
```

วิธีรันการทดสอบ Playwright ในโหมด headed:

```bash
npx playwright test --headed
```

## ตัวแปรสภาพแวดล้อม

| ตัวแปร | ค่าเริ่มต้น | คำอธิบาย |
|----------|---------|-------------|
| `ENABLE_VNC` | `true` | ตั้งค่าเป็น `false` เพื่อปิดใช้งานชุด VNC |
| `VNC_RESOLUTION` | `1280x1024x24` | ความละเอียดของจอแสดงผลเสมือน (กว้างxสูงxความลึกสี) |
| `DISPLAY` | `:99` | หมายเลขจอแสดงผล X11 |
| `NOVNC_HOST_PORT` | `6080` | พอร์ตฝั่งโฮสต์สำหรับ noVNC (เปลี่ยนเพื่อหลีกเลี่ยงความขัดแย้งกับคอนเทนเนอร์อื่น) |

ตั้งค่าเหล่านี้ในไฟล์ `.env` ของคุณหรือใน `.devcontainer/devcontainer.json` ภายใต้ `containerEnv`

## การปิดใช้งานชุด VNC

สำหรับการทำงานแบบ headless เท่านั้น ให้ตั้งค่า `ENABLE_VNC=false` ใน `.env` และรีสตาร์ทคอนเทนเนอร์:

<TabItem label="Docker">

```bash

docker compose up -d
```

</TabItem>

