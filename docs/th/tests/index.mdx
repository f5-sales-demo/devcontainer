การทดสอบเหล่านี้ตรวจสอบว่าระบบย่อมหลักของ devcontainer ทำงานร่วมกันได้ในขณะรันไทม์ การทดสอบเหล่านี้ไม่ได้ฝังอยู่ใน Docker image — ให้รันภายในคอนเทนเนอร์ที่กำลังทำงานอยู่เพื่อตรวจสอบความสามารถของมัน

## การทดสอบที่มี

| การทดสอบ | สิ่งที่พิสูจน์ได้ |
| ---- | ---- |
| [การบันทึกหน้าจอ](/devcontainer/tests/screen-recording/) | Xvfb display, Chromium แบบมีหน้าจอ (Playwright) และการบันทึกด้วย ffmpeg สร้างไฟล์ MP4 ที่ถูกต้อง |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | Chrome symlink, headless patch และการนำทางผ่าน MCP สร้างภาพหน้าจอที่ถูกต้อง |
