การทดสอบเหล่านี้ตรวจสอบว่าระบบย่อมหลักของ devcontainer ทำงานร่วมกันได้ในขณะรันไทม์ การทดสอบเหล่านี้ไม่ได้ถูกฝังอยู่ใน Docker image — ให้รันภายในคอนเทนเนอร์ที่กำลังทำงานอยู่เพื่อตรวจสอบความสามารถของมัน

## การทดสอบที่มีให้ใช้งาน

| การทดสอบ | สิ่งที่พิสูจน์ได้ |
| ---- | ---- |
| [การบันทึกหน้าจอ](/devcontainer/tests/screen-recording/) | จอแสดงผล Xvfb, Chromium แบบมีหน้าจอ (Playwright) และการบันทึกด้วย ffmpeg สร้างไฟล์ MP4 ที่ถูกต้อง |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | Symlink ของ Chrome, แพตช์แบบ headless และการนำทางผ่าน MCP สร้างภาพหน้าจอที่ถูกต้อง |
