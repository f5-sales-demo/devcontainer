これらのテストは、devcontainer の主要なサブシステムがランタイム時に連携して動作することを検証します。Docker イメージに組み込まれているわけではありません — 実行中のコンテナ内で実行して、その機能を確認してください。

## 利用可能なテスト

| テスト | 検証内容 |
| ---- | ---- |
| [画面録画](/devcontainer/tests/screen-recording/) | Xvfb ディスプレイ、ヘッド付き Chromium（Playwright）、および ffmpeg 録画が有効な MP4 を生成すること |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | Chrome シンボリックリンク、ヘッドレスパッチ、および MCP ナビゲーションが有効なスクリーンショットを生成すること |
