これらのテストは、devcontainerの主要なサブシステムが実行時に連携して動作することを検証します。Dockerイメージには組み込まれていません。実行中のコンテナ内で実行して、その機能を確認してください。

## 利用可能なテスト

| テスト | 検証内容 |
| ---- | ---- |
| [画面録画](/devcontainer/tests/screen-recording/) | Xvfbディスプレイ、ヘッド付きChromium（Playwright）、およびffmpeg録画が有効なMP4を生成すること |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | Chromeシンボリックリンク、ヘッドレスパッチ、およびMCPナビゲーションが有効なスクリーンショットを生成すること |
