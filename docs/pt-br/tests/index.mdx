Esses testes validam que os principais subsistemas do devcontainer funcionam em conjunto em tempo de execução. Eles não são incorporados na imagem Docker — execute-os dentro de um contêiner em execução para verificar suas capacidades.

## Testes Disponíveis

| Teste | O que comprova |
| ---- | ---- |
| [Gravação de Tela](/devcontainer/tests/screen-recording/) | O display Xvfb, o Chromium com interface gráfica (Playwright) e a gravação com ffmpeg produzem um MP4 válido |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | O symlink do Chrome, o patch headless e a navegação MCP produzem uma captura de tela válida |
