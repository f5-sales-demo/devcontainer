ये परीक्षण सत्यापित करते हैं कि devcontainer के प्रमुख उपतंत्र रनटाइम पर एक साथ कार्य करते हैं। ये Docker इमेज में अंतर्निहित नहीं हैं — इसकी क्षमताओं को सत्यापित करने के लिए इन्हें एक चालू कंटेनर के अंदर चलाएँ।

## उपलब्ध परीक्षण

| परीक्षण | यह क्या प्रमाणित करता है |
| ---- | ---- |
| [स्क्रीन रिकॉर्डिंग](/devcontainer/tests/screen-recording/) | Xvfb डिस्प्ले, हेडेड Chromium (Playwright), और ffmpeg रिकॉर्डिंग एक वैध MP4 उत्पन्न करते हैं |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | Chrome सिमलिंक, हेडलेस पैच, और MCP नेविगेशन एक वैध स्क्रीनशॉट उत्पन्न करते हैं |
