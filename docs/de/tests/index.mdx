Diese Tests überprüfen, ob die wichtigsten Subsysteme des Devcontainers zur Laufzeit zusammenarbeiten. Sie sind nicht in das Docker-Image integriert — führen Sie sie in einem laufenden Container aus, um dessen Fähigkeiten zu verifizieren.

## Verfügbare Tests

| Test | Was er beweist |
| ---- | ---- |
| [Bildschirmaufzeichnung](/devcontainer/tests/screen-recording/) | Xvfb-Display, Chromium mit Benutzeroberfläche (Playwright) und ffmpeg-Aufzeichnung erzeugen eine gültige MP4-Datei |
| [Chrome DevTools MCP](/devcontainer/tests/chrome-devtools-mcp/) | Chrome-Symlink, Headless-Patch und MCP-Navigation erzeugen einen gültigen Screenshot |
